# BreakController

## Скачивание готового приложения

Последний релиз доступен в разделе [Releases](https://github.com/solova3930/BreakController/releases).  
Просто скачайте `PCRestTimer.exe` и запустите.

Программа для планирования и контроля перерывов пользователей ПК. Позволяет автоматически блокировать ввод, показывать напоминания об отдыхе и управлять рабочим временем пользователя.

## Описание

Проект представляет собой Windows Forms приложение на C# с основным классом `BreakController`, который отвечает за:

- Запуск таймеров работы и отдыха.
- Блокировку клавиатуры и мыши во время отдыха.
- Отображение форм с уведомлением о перерыве.
- Управление пользовательским выбором (отметка о начале/окончании работы).

Приложение помогает пользователям следовать режиму работы и отдыха, снижая усталость и повышая продуктивность.

## Структура проекта
/BreakController
│

├─ Form1.cs                  # Главное окно приложения
├─ Program.cs                # Точка входа
├─ BreakController.sln       # Решение Visual Studio
├─ README.md                 # Этот файл
└─ .gitignore                # Исключает bin/, obj/ и временные файлы
Особенности
Таймеры работы и отдыха настраиваются через настройки.
В период отдыха ввод с клавиатуры и мыши блокируется.
Пользователь видит уведомления о необходимости перерыва.
Программа не требует дополнительных библиотек.
